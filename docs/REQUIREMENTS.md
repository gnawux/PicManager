我们要做一个功能完善、易于管理、确保照片和相关数据安全的家庭照片与图片管理工具。

## 核心功能

* 支持自动整理照片，提取时间、地点信息、拍摄相机信息；
* 支持重复图片识别，包括图片尺寸缩放后的重复照片，去重功能需要人工确认后再删除；
* 按照时间和地点、拍摄相机划分相册，可以手工合并；

## 格式支持

* 支持多种格式，包括 JPG，索尼的 RAW 格式，苹果的动态照片，网上常见的 webp, png, gif 等格式

## 输入输出

* 从指定目录导入照片文件
* 其他导入方式后续考虑

## 导入行为

* **缺省行为**：将照片 **移动**（move）到 library 中，按拍摄日期放入 `yyyy-mm-dd` 子目录，导入完成后源文件不再保留
* **保留原文件**：提供 `--copy` 参数，改为复制而不删除源文件；适合导入重要原始存档时使用
* **拍摄日期推断优先级**（从高到低）：
  1. EXIF DateTimeOriginal（最可靠）
  2. 文件名推断：文件名中包含日期时间模式（如 `IMG_20240615_103000.jpg`、`2024-06-15_photo.jpg`），或文件名整体为 Unix 时间戳（秒级或毫秒级）
  3. 无法判断：照片放入 `unknown/` 子目录，后续通过编辑拍摄时间将其重新归入正确日期目录
* 重复导入同一文件（相同 SHA-256）时跳过，不重复移动或复制

## 实现偏好

* 用 Rust 实现
* 提供 Web 操作界面和简单的命令行界面

## 其他要求

* 导入后照片在 library 中按 `yyyy-mm-dd` 目录组织，数据库记录照片在 library 内的新路径
* 运行在 Mac 平台上，未来考虑其他平台
