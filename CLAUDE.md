# PicManager

家庭照片与图片管理工具。支持自动整理、去重、按时间/地点/相机分相册，提供 Web 界面和命令行界面。

## 技术栈

- 语言：Rust
- 界面：Web UI（前端 + Rust 后端 HTTP server）+ CLI
- 目标平台：macOS（未来扩展其他平台）

## 核心功能模块

1. **导入**：从指定目录扫描并导入照片，记录导入状态（已导入/未导入），方便用户手工删除源文件
2. **元数据提取**：从 EXIF/元数据中提取拍摄时间、GPS 地点、相机型号
3. **去重**：识别重复图片（含缩放后的重复），去重操作需人工确认后才执行删除
4. **相册组织**：按时间、地点、拍摄相机自动划分相册，支持手工合并相册

## 支持格式

- JPG/JPEG
- 索尼 RAW（ARW）
- 苹果动态照片（HEIC/Live Photo）
- WebP、PNG、GIF

## 项目结构约定

```
src/
  main.rs          # CLI 入口
  lib.rs           # 库根
  importer/        # 导入模块
  metadata/        # 元数据提取
  dedup/           # 去重逻辑
  album/           # 相册管理
  web/             # Web 服务器和 API
    handlers/
    routes.rs
  db/              # 数据存储层
```

## 开发原则

- 照片和数据安全优先：删除操作必须有确认步骤，原始文件不做破坏性修改
- 去重删除需人工确认，程序只标记，不自动删除
- 导入后保留导入状态记录，方便用户手工清理源目录
- 数据库用于存储元数据和状态，照片文件本身不做格式转换
